### APP Tempo agora desenvolvido em .NET MAUI

#### Projeto feito nas aulas da Fatec jahu em dispositivos móveis, nele, aprendemos a consumir uma API externa gratuita, a openwether, e também lidamos com geolocalização, funções nativas .NET.

#### FEATURES DO PROJETO
- consumo de API externa
- possibilidade de buscar localização do usuário*
- Busca do clima de qualquer cidade do mundo
- Iframe retornando as imagens meteorológica da cidade e arredores

*Para windows machine, emulador virtual, as permissões para acesso de localização já foram aceitas, caso queira testar no próprio celular, por favor, aceite as permissões que o Maui pedir.
